업종별 견적서 쓰는 법 — 디자인·개발·시공·촬영

품목 쪼개는 법 · 업종별 필수 명시 조건 · 실전 예시

견적서의 기본 구조는 어느 업종이든 같지만, 품목을 어떻게 쪼개고 어떤 조건을 명시하느냐는 업종마다 완전히 다릅니다. 그리고 분쟁은 대부분 그 업종 특유의 "안 적어둔 것"에서 터집니다. 디자인은 수정 횟수, 개발은 범위 변경, 시공은 자재와 추가 공사, 촬영은 원본 제공 여부가 그렇습니다. 업종별로 하나씩 살펴봅니다.

디자인 (로고·브랜딩·편집·웹디자인)

디자인 견적의 핵심은 수정 범위를 품목에 녹이는 것입니다. "로고 디자인 일체 80만원"이라고 쓰면 수정 10번도 그 금액에 포함된 것처럼 보입니다.

품명규격수량단가
로고 디자인 (시안 3종)AI·PNG 납품1800,000
수정 (3회 포함 후 회당)경미한 수정 기준-50,000
명함 디자인양면1150,000

비고란에 꼭 쓸 것: 수정 몇 회까지 무료인지와 "경미한 수정"의 정의(색·글자 교체는 수정, 컨셉 변경은 신규), 원본 파일(AI 등) 제공 여부와 추가 비용, 저작권·사용 범위(로고는 양도, 포트폴리오 게재 허용 등), 인쇄비 포함 여부.

개발 (웹사이트·앱·시스템)

개발 견적의 적은 범위 변경(스코프 크리프)입니다. "홈페이지 제작 300만원" 한 줄짜리 견적은 기능이 하나씩 늘어나는 것을 막지 못합니다. 기능 단위로 쪼개세요.

품명규격수량단가
메인 페이지 퍼블리싱반응형1800,000
서브 페이지반응형5200,000
문의 폼 + 관리자 알림이메일 연동1300,000
유지보수 (선택)월 단위-150,000

비고란에 꼭 쓸 것: 견적에 포함되지 않은 것(호스팅·도메인 비용, 콘텐츠 원고 작성, 기획 변경에 따른 재작업), 범위 외 기능 추가 시 별도 견적이라는 문구, 납기와 검수 기간, 하자보수 기간(예: 오픈 후 30일 무상).

시공 (인테리어·설비·수리)

시공 견적은 자재비와 인건비를 분리하고, 자재의 등급을 명시하는 것이 기본입니다. "욕실 리모델링 350만원"이 아니라 자재·품목별로 풀어 써야 나중에 "이 타일이 아닌데요" 분쟁을 막을 수 있습니다.

품명규격수량단가
바닥 타일 (자재)OO브랜드 600각12㎡35,000
타일 시공 (인건)철거 포함2일350,000
양변기 교체OO모델 기준1280,000
폐기물 처리1톤 기준1150,000

비고란에 꼭 쓸 것: 자재 브랜드·모델 변경 시 차액 정산 방식, 철거 후 발견되는 추가 공사 (누수·배관 등)는 별도 협의라는 문구, 공사 기간과 지연 시 처리, 잔금 지급 시점(준공 확인 후).

촬영·영상 (사진·홍보영상·행사)

촬영 견적의 단골 분쟁은 원본 제공과 수정 횟수입니다. 보정본 몇 컷, 원본(RAW) 제공 여부, 영상 수정 몇 회를 반드시 품목이나 비고에 명시하세요.

품명규격수량단가
제품 촬영보정본 20컷1회500,000
추가 보정컷당-20,000
홍보영상 편집60초 내외, 수정 2회 포함1900,000

비고란에 꼭 쓸 것: 원본(RAW/촬영본) 제공 여부와 비용, 수정 횟수 초과 시 단가, 촬영 연기· 취소 규정(우천 등), 결과물 사용 범위(광고 집행 여부에 따라 단가가 달라지는 경우), 출장비.

업종 불문 공통 원칙 4가지

  1. 품목은 "일체"라는 말 대신 쪼개서 — 쪼갤수록 협상도 쉬워집니다. 고객이 예산을 줄이고 싶어 하면 품목 단위로 빼면 되니까요.
  2. 포함 안 되는 것을 적는 게 진짜 실력 — 포함 목록보다 "불포함:" 한 줄이 분쟁을 더 많이 막습니다.
  3. 지급 조건은 숫자로 — "계약금 50%, 잔금은 납품 후 7일 이내"처럼 비율과 기한을 명시. 자세한 항목 구성은 견적서 작성법 가이드 참고.
  4. 부가세 표기 확인 — 별도인지 포함인지 반드시 명시. 부가세 별도 vs 포함에서 정리했습니다.
